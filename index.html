Diferencias entre lenguaje HTML y CSS

El HTML y el CSS son dos lenguajes fundamentales y complementarios para crear páginas web , pero tienen funciones distintas. Primero, el HTML especifica el qué (contenido)- la estructura de una página web y el CSS especifica el cómo (estilo y diseño) - mostrar el contenido de una página web. Están pensados para trabajar juntos pero de forma separada, lo que permite cambiar fácilmente el diseño sin afectar el contenido.

Onshape

En esta clase usamos un nuevo programa el cual es Onshape. Esta es una plataforma de desarrollo de productos en la nube que ofrece capacidades de CAD (Diseño Asistido por Computadora) y PDM (Gestión de Datos de Producto). Permite a los usuarios crear , editar y colaborar en modelos y diseños en 3D a través de un navegador web, sin necesidad de instalaciones ni descargas. Github y Netlify

GitHub es una plataforma de alojamiento de código fuente y control de versiones, mientras que Netlify se especializa en el despliegue y la gestión automatizada de sitios web estáticos, ofreciendo funcionalidades adicionales para mejorar la experiencia de desarrollo web. En muchos casos, ambos servicios se utilizan de manera complementaria: el código fuente se almacena en GitHub, y Netlify se utiliza para implementar y alojar el sitio web basado en ese código.

Rotación (motor CC - Arduino)

La rotación de un motor de corriente continua (CC) con un Arduino implica la conexión del motor al Arduino mediante pines de salida digital y el control de la dirección y velocidad utilizando un puente H. El puente H permite invertir la polaridad para cambiar la dirección de rotación. La velocidad se controla mediante modulación de ancho de pulso (PWM). La programación en Arduino utiliza funciones para activar los pines de control y ajustar la velocidad, lo que permite dirigir el motor en sentido horario o antihorario, detenerlo y variar su velocidad de manera programática.

Sensor con luz led

El proyecto combina un sensor ultrasónico HC-SR04 con una placa Arduino y luces LED para crear un sistema interactivo de iluminación. El sensor ultrasónico mide la distancia a un objeto, y según esta distancia, se controla el brillo de las luces LED. Cuando un objeto se acerca a menos de 30 centímetros, las luces LED se encienden gradualmente, y cuando el objeto se aleja, las luces se apagan de manera gradual. Este comportamiento se logra mediante el uso de pines digitales de entrada y salida en la placa Arduino, junto con el control de intensidad luminosa a través de la función *analogWrite()*.

Sensor de temperatura

El proyecto integra un sensor de temperatura con una placa Arduino y luces LED para crear un sistema de iluminación basado en la temperatura ambiente. El sensor de temperatura mide la temperatura del entorno y la placa Arduino utiliza esta información para controlar el brillo de las luces LED. Por ejemplo, podría programarse para que las luces se vuelvan más intensas a medida que la temperatura aumenta y se atenúen cuando la temperatura disminuye. Este enfoque permite una visualización inmediata de las variaciones de temperatura, convirtiendo datos de sensores en respuestas visuales a través de la programación en Arduino.

Pseudocódigo

representación intermedia entre un lenguaje de programación y un lenguaje natural que se utiliza para describir la lógica de un algoritmo o programa. No sigue una sintaxis estricta como un lenguaje de programación real, pero proporciona una forma estructurada de expresar la secuencia de pasos necesarios para realizar una tarea específica. Su objetivo principal es facilitar la comprensión y la planificación de algoritmos antes de la implementación en un lenguaje de programación específico.
