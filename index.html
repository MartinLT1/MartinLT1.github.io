Diferencias entre lenguaje HTML y CSS

El HTML y el CSS son dos lenguajes fundamentales y complementarios para crear páginas web , pero tienen funciones distintas. Primero, el HTML especifica el qué (contenido)- la estructura de una página web y el CSS especifica el cómo (estilo y diseño) - mostrar el contenido de una página web. Están pensados para trabajar juntos pero de forma separada, lo que permite cambiar fácilmente el diseño sin afectar el contenido.

Onshape

En esta clase usamos un nuevo programa el cual es Onshape. Esta es una plataforma de desarrollo de productos en la nube que ofrece capacidades de CAD (Diseño Asistido por Computadora) y PDM (Gestión de Datos de Producto). Permite a los usuarios crear , editar y colaborar en modelos y diseños en 3D a través de un navegador web, sin necesidad de instalaciones ni descargas. Onshape ofrece características como actualizaciones automáticas, CAD estable y confiable, PDM integrado y la capacidad de acceder a proyectos de CAD desde cualquier dispositivo.

Proyecto "SPY KEY"

Prototipado

El producto elegido para Laboratorio # 2 es "SPY KEY", este proyecto es una combinación entre una cámara espía y una llave de seguridad. No es para usos exagerados o protectores al 100% como una caja fuerte sino, para un uso más casual como guardar una agenda, dulces o algo importante pero que no tenga un valor estrambótico como las joyas u otros derivados, ya que no está dirigido para ese uso en general.
